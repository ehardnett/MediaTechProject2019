The IP routing fabric that these control rooms sit on allows for an immense number of resources to be shared flexibly and dynamically between spaces, so that control rooms can grow and shrink in size as needed to best serve productions. All 18 production positions in each room have dedicated touch screen control panels, so that producers can control their own lighting and monitoring to suit their preferences.
